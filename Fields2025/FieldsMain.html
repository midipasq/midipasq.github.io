Recommended Reading
  • The algebra of splines: duality, group actions, and homology . Survey article on splines and the recommended reading for our focus group.
Other articles we may reference
  • Generalized Splines on Arbitrary Graphs . In this article the setup for generalized splines on graphs -- which we plan to use -- was first explained.
  • Module Bases for Multivariate Splines . A generalization of Saito's criterion from hyperplane arrangements to the context of splines.
  • The Euler multiplicity and addition-deletion theorems for multiarrangements. It would be great to find addition-deletion theorems for splines that mirror what has been done for multiarrangements.

How to run .md files in the Macaulay2 web interface
  1. Download the .md file to your computer (make sure to save it with the .md file extension)
  2. Open the Macaulay2 web interface at this link .
  3. In the left-hand panel, scroll down and click on the "Load your own tutorial" tab.
  4. When prompted, upload the .md file that you just downloaded.
  5. You should now see the uploaded file appear as a new tab on the left hand side.
