Getting Started in Macaulay2
We encourage everyone to set up a CoCalc account: please go to https://cocalc.com/ to set up an account. Ideally, we can make five or six collaboration groups in CoCalc. To use Macaulay2 in CoCalc follow these instructions:
  • Create a project
  • Click the "New" button
  • Click "Linux Terminal"
  • Inside the linux terminal, type "M2" and press enter
Alternatively, you can try Macaulay2 online! . This option appears to not work currently - if it starts working we will make a remark here.

Finally, if you wish to install Macaulay2 on your computer, go to https://faculty.math.illinois.edu/Macaulay2/Downloads/ . Make sure to follow the instructions that match your operating system. It will probably take several tries to get everything installed and working properly (especially on Windows!). Be patient, and don't hesitate to ask for help!
